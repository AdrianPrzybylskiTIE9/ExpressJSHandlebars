# Makaroniarze Handlebars

## 🍝 O projekcie

"Makaroniarze" to strona internetowa z przepisami na dania makaronowe. Aplikacja została stworzona przy użyciu NodeJS, Express i Express Handlebars.

## 💻 Języki i API

- NodeJS
- Express
- Express Handlebars
- JQuery
